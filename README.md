## Nix Bootstrap

Clone this repo anywhere, then run the bootstrap script from the repo root:

```bash
./bootstrap.sh mac
```

or

```bash
./bootstrap.sh linux
```

The script uses the current checkout path, so the repo no longer needs to live at `~/Dev/dotfiles`.
